Node

Node.run

Starts the event loop for async/sync operations.

Type definition

let run: unit => unit;

Usage

open LwtNode;

Node.run({
  let%lwt myDir = Fs.mkdir("myDir");
  let%lwt myDir2 = Fs.mkdir("myDir2");
  Node.resolved();
});
